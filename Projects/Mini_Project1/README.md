###                                   ShopEZ - Frontend 

ShopEZ is a simple e-commerce frontend built using HTML, CSS, Bootstrap, and JavaScript.
It displays top deals, allows product search and category filtering, and shows products dynamically.

##  Features

-> Register/login

-> Responsive Bootstrap Navbar

-> Carousel (Top Deals) with auto-slide

-> Search functionality for products

-> Category filtering (Electronics, Fashion)

-> Dynamic Product Grid display

-> Validations(cart+checkout)

-> Modular JavaScript structure

-> Footer section

##  Technologies Used

-> HTML5

-> CSS3

-> Bootstrap 5

-> JavaScript (JS + jQuery)

## How to Run

-> Download or clone the project

-> Open the project folder

-> Run index.html in your browser
